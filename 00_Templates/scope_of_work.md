# Scope of Work

## Project

Ecommerce Data-to-Dashboard Automation Sprint.

## Goal

Build an auto-updating dashboard that helps the client track revenue, profit, orders, product performance, and ad return without manually combining spreadsheets every week.

## Included

### Starter - 600 USD

- One Power BI dashboard page.
- One data source, such as Salla, Zid, Shopify export, Google Sheet, or CSV.
- Basic cleaning pipeline.
- Core KPIs: revenue, orders, profit, average order value.
- One handover call or recorded walkthrough.

### Standard - 900 USD

- Two Power BI dashboard pages.
- API or scheduled data connection.
- One additional source, such as ad spend or product cost.
- Core KPIs plus ROAS, product ranking, weekly growth.
- Simple documentation PDF.

### Pro - 1,200 USD

- Three Power BI dashboard pages.
- Scheduled refresh flow through Power Automate, Apps Script, or equivalent.
- Weekly PDF report setup where technically available.
- More detailed product, order, and ad analysis.
- Simple documentation PDF plus 7 days post-delivery support.

## Not Included

- Redesigning the client's store.
- Fixing the client's accounting system.
- Manual data entry beyond the sample needed for setup.
- Custom CRM or ERP development.
- New dashboard pages after scope approval.
- Ongoing maintenance unless a monthly retainer is agreed.

## Payment Terms

- 50 percent upfront before work starts.
- 50 percent before final dashboard access is released.
- Extra work is quoted separately before implementation.

## Client Responsibilities

- Provide access keys, exports, or sample files within 24 hours.
- Confirm KPI definitions, such as profit and cost, before dashboard build.
- Review the dashboard within 48 hours of delivery.

## Delivery Assets

- Power BI file or Power BI Service link.
- Cleaned dataset or connected data source.
- Automation script or flow summary.
- Short documentation file.
