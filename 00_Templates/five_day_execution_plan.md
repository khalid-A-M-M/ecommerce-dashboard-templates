# Five-Day Execution Plan

## Day 1 - Extract

- Confirm access and sample data.
- Pull orders, products, costs, and ad spend where available.
- Save raw data in the client folder.
- Document missing fields immediately.

## Day 2 - Clean

- Standardize dates.
- Remove zero-revenue, cancelled, and refunded rows.
- Normalize product names.
- Add profit fields.
- Export dashboard-ready CSV or table.

## Day 3 - Model

- Import cleaned data into Power BI.
- Create date table.
- Create product table if needed.
- Build relationships.
- Add core DAX measures.

## Day 4 - Dashboard

- Build agreed pages.
- Add filters for date, product, and status.
- Check readability.
- Compare totals against the raw sample.

## Day 5 - Deliver

- Test refresh.
- Prepare PDF documentation.
- Send final invoice.
- Release dashboard access after payment.
- Offer monthly retainer.
