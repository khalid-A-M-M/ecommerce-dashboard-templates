# Delivery Checklist

## Before Final Delivery

- [ ] Data source connection works.
- [ ] Refresh test completed successfully.
- [ ] Dates are parsed correctly.
- [ ] Revenue, cost, and profit totals match client sample.
- [ ] Cancelled or refunded orders are handled.
- [ ] Product names are standardized.
- [ ] Dashboard pages match agreed scope.
- [ ] DAX measures are checked.
- [ ] Filters and slicers work.
- [ ] Dashboard is readable on the client's screen size.
- [ ] Documentation file is complete.
- [ ] Client repository or code folder is ready.
- [ ] Final invoice is sent.
- [ ] Final access is released after payment.
