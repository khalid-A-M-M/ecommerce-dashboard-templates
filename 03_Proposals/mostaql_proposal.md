# Mostaql Proposal Template

السلام عليكم،

قرأت طلبك بخصوص تقارير المبيعات وربط البيانات. أقدر أساعدك في تحويل بيانات المتجر إلى لوحة Power BI واضحة تتحدث تلقائياً، بدل تجميع الأرقام يدوياً من أكثر من ملف.

ما سأقدمه:

- ربط مصدر البيانات المتاح، سواء API أو Google Sheet أو Excel.
- تنظيف بيانات الطلبات والمنتجات والتكاليف.
- بناء مؤشرات أساسية مثل الإيرادات، الأرباح، عدد الطلبات، متوسط قيمة الطلب، وأفضل المنتجات.
- تصميم لوحة عملية سهلة القراءة.
- تسليم شرح مختصر لطريقة قراءة اللوحة وتحديثها.

طريقة العمل تكون خلال 5 أيام:

اليوم 1: استلام البيانات وتجهيز الربط.

اليوم 2: تنظيف البيانات وتجهيزها.

اليوم 3: بناء نموذج Power BI والمقاييس.

اليوم 4: تصميم اللوحة.

اليوم 5: اختبار التحديث والتسليم.

أحتاج منك في البداية معرفة منصة المتجر ومصدر بيانات الطلبات، وهل لديك تكلفة المنتجات وبيانات الإعلانات أم لا.

تحياتي.
